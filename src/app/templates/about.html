О проекте Vacsh

Vacsh — сервис для автоматизации откликов на вакансии. Загрузите резюме, подтвердите данные, выберите сайт для поиска работы и получите подборку подходящих вакансий с помощью искусственного интеллекта.

Сейчас поддерживается поиск по hh.ru. В будущем планируется поддержка других платформ.
